## Simple Flicker Photo Search & Viewer
- Use ES2015, React, BABEL, Webpack

![Screenshot](/scr.png)

### How to run locally

```
git clone git@github.com:yhagio/react-flickr.git
npm install 
npm install webpack -g
webpack -w
```
Open `index.html` in browser. Voila!
